program description
===================
  The program is aimed to validate various peripherals and connectors on
MB913 board:

- 8M Hz HSE crystal (X2)
- 32k Hz LSE crystal (X3)
- Buttons: B1, B2
- 2 color LEDs: LD3,LD4
- Embedded ST-Link (via CN1) & LD1


Hardware environment
====================
- MB913 Rev.B board with STM32F100RBT6B on board

Software environment
====================
- IAR EWARM V5.50 (or higher)
- STM32F100(Valueline) FW Library V0.1 (or higher)
